天涯清露集【议精】
词：

[ 虞美人. 春樱]

试香埋玉今何世，春在斜阳地。半销英气半同尘，不道红愁成海块成云。

吾心尝与东风借，吾鬓星霜迹。看花开到最深情，坐到身无余恨感无名。

 

[ 虞美人.樱花七日]

红荒艳冷春无极，陡识人间仄。离襟游屐驻微香，剩有华年胸次近斜阳。

长风去去生奇悔，七日摘星蕊。从知摽落判畸零，负此一期一会一深情。

 

[ 虞美人.赋得家中水仙新开]

春愁相值还相误，况满身清露。天涯明日又东风，不道小窗霜鬓语玲珑。

对花永夜如观己，心事终难恃。人间岁月去堂堂，且放青青一萼到生香。

 

[ 虞美人.岛君赠玫瑰，时值情人节]

夏花幽独秋花淡，夜永飞霜满。故人遗我以春花，为祝湖山有约鬓无华。

馨香愿及芳年赏，费尽东风想。开成清曙到轻红，不道苍茫落照是深衷。

 

[ 菩萨蛮.栀花一组]

风期十载春衫旅，花明一眼青芜雨。清瑟最堪亲，孤华夜气新。

别轻情的的，香烈传衷臆。尔我正相忘，如何思沛滂。

 

一花拭露弹新泪，一花烛夜销浮岁。磊落结同心，都曾著意深。

栖香愁未稳，影事重头认。江海故人思，中宵月到时。

 

只星天末荧荧坠，有人衣角煌煌佩。流眄互芬芳，望中岁月长。

暌离心事仄，忆作辉容隔。鬓影驻东风，与花俱梦中。

 

珠光照乘如初识，馨魂终不闻霜息。直是累花身，愔愔住世尘。

折荣空致远，容有三生愿。相遗道绵绵，江南落萼天。

 

[ 菩萨蛮.次朵君韵咏莫愁湖上遛荷叶旧事]

烟华浓觅伤心绿，波声流寂《江城》曲。去棹复回看，六朝风色残。

千情脉脉了，一水盈盈好。霜鬓与尘丝，斜晖红不知。

 

[ 菩萨蛮.玉兰新发]

平明花事昭昭发，中宵花是明明月。不悔旧吹香，搴芳安可忘。

且留尘梦老，此外皆清好。但使愿无违，打头花正肥。

 

[ 菩萨蛮.寄人一组]

羁栖郁郁知无待，薄游了了疑沧海。露世与年涯，瞢腾说梦华。

劳生歌独夜，吾道艰虞也。心上起风波，绝知怀汝多。

 

风流早为幽人绝，诗中春气连年竭。远意尚拳拳，余欢销未全。

万殊容鬓老，一样忧端少。声气亦相依，如何负素期。

 

斯年迫忆霓虹冷，斯人孤往江声永。风雨壮离思，倘容岁月滋。

存谁肝胆古，去者朝和暮。何以致青春，斯言终未陈。

 

[ 满江红.  癸巳七夕]

花不堪持，星堪语、东风楼上。人天有、古愁今恨，众生梦想。良愿固违肝胆冷，沉歌早负诗词壮。者迢遥一水复千山，曾相望。

双鬓老，一襟放。衔薄慨，销奇怅。识中年哀乐，红尘板荡。夜雨江湖吾痼疾，海桑世界君无恙。又何须一夕尽千言，从头谅。

 

[ 满江红.甲午七夕]

白雨飞楼，终隔断、天涯弥望。当此夕、梦君颜色，与君扺掌。有约湖山堪下泪，无情天意高难想。祗尘中清忆醉中歌，空悲壮。

眉萼损，愁根长。今非昨，身同妄。愿双星照见，三生业障。未忍馨香辞故蕊，转头听尽新荷响。羡东风管得并枝花，犹相向。

 

[ 满江红.乙未七夕]

星渚云涯，知几度、迍邅孤往。曾会得、清歌似玉，谓谁河广。世事难同尘鬓白，情天且共心旌绛。正流年杯酒动渊思，吾言妄。

“ 占熊梦，劳魂想。夜雨约，山海障。愿名花在侧，明珠擎掌。”恩拭镜文光未灭，风临平旦愁先莽。纵休休莫莫到他生，双双忘。

 

[ 减兰. 次城城赋兰花新开韵]

扬扬香沸，天劲秋浓花未睡。风撼衣痕，浩露续愁入鬓唇。    

眼情心事，宵永容消湖海气。月正明时，自怅斯人纫较迟。 

 

[ 应天长. 茉莉]

夜声清绽头边雪，小萼初香宵欲裂。和衣起，和愁撷，万一东风肝胆竭。   

古相思，秋可掇，指隙一痕霜屑。我已明明似月，如何未化蝶。

 

[ 临江仙 . 次城城韵同题咏梅]

欲振满衣星露色，鬓间孤白成云。不辞香怒破宵昏，我怜花是梦，花觑我同尘。

小萼冲寒烟水外，东风犹峙重门。三千世界一枝春，情愁开且落，忏得几销魂。

 

[ 临江仙.叠旧韵再咏梅]

手握馨香终远志，胸中磊块如云。怅从花古认寰昏。累君霜雪鬓，住此软红尘。

尘事妥还芳事叵，立成明月当门。一生幽契十方春。华枝真可冀，无梦感精魂。

 

[ 临江仙.岁末忆及与友同游玄武湖事] 

遥向羁年追旧迹，化虹化月行藏。化缁衣上迸初霜，风波离思满，容鬓老荷苍。

胸臆横秋悲未已，眼中私涌春芒。天涯壮旅路歧长，沧桑余一瞥，四子驻流光。

 

[ 金缕曲. 途见木芙蓉]

露下泠泠意。负秾华、悭朱酽白，晚香微蕊。搴木西风江湖远，慨想红尘积毁。销已尽、半生英气。欲语终朝空望极，更繁当愁破秋当秽。昔时月，吾能罪。

耿相忆是终相累。敛余妍、峙身同妄，持心犹慧。清瑟一枝成孤倚，永谢岁滋花坠。酬天末、故人情味。素抱依然霜词老，到前缘后迹俱生泪。今夕雨，吾应醉。

 

[ 千秋岁引.寄人]

海上涛云，京华尘土，山色青销旧眉妩。斜阳小立余秋鬓，浮生大梦倾春醑。浅相思，深相语，不相遇。

忆共梅花相尔汝，忆共东风相怨慕。十载天涯与清露。昨宵长是春相负，来朝犹是春相付。故人心，催人老，留人住。

 

[ 子夜歌.春]

别星萍故人老未，得似十年前俊。濯衣尘终朝泪冷，早识露零霜峻。清酌沉宵，空楼微雨，破寂寒成阵。怅暌离云树春程，云水旧盟不辨，梦华情恨。 

待愁祓天涯节序，遥嗣涉江音信。词到孤衷，歌犹况味，祗记盈盈问：便千红过眼，襟期明日能准。予意绸缪，一枝栖定，心与斜阳近。且凭肩留语东风，放花吹鬓。

 

[ 子夜歌.夏]

隔容辉旧游疏旷，慨想年涯羁束。拥飞光尘欢孤负，危梦故人眉目。销鬓成丝，遣身分影，世网留三宿。记沉沉风露初衷，终夕披襟秉烛，一杯相属。

且兼忘吹花题叶，嗟彼人天幽独。漠漠愁城，渊渊寰海，素愿增踟蹰。况天涯挥袂，望迷身世南北。君意如鸿，我心匪席，长患情能卜。恨暌携虚约西湖，万荷摇绿。

 

[ 河传.白芍药]

新月，初雪，想清晖。相对一枝一卮，相看开时摇落时。将离，春风起路歧。

远道客裳犹露迹，值梦夕，贪此隔花忆。是绸缪，是温柔，是愁，如吾者白头。

 

[ 河传.玉兰]

云萼，风约，最盈盈。玉想琼思一生，清泪狂愁何纵横。杯停，昔年意气倾。

吾辈宁辞浮大白，香郁积，竟夕春同寂。小莲花，不归家，祗嗟，新添几鬓华。

 

[ 河传.海棠]

幽态，奇彩，思无邪。英气空销岁华，恃芳眷言蝶恋花。嗟嗟，那枝红一些。

月不分明花寤寐，花有泪，花是平生事。断相闻，三十春，美人，眉头见着尘。

 

[ 蝶恋花. 十年]

云水荒荒山漫漫，银浦红桑，一念飞光换。别梦星霜成荏苒，春心泥絮无由判。

怀远可堪容泪满，世味衷情，两两耽肝胆。影事十年风雨半，莫知眉妩和眉淡。

 

何必忘形春酌里，我有襟期，樽有樱纷委。意蕊心香皆块垒，歌怀酒绪空狂慧。

万事能遗犹热耳，溯尽风波，此意寻常尔。百醆不辞终夕醉，十年且识愁滋味。

   

各向天涯为壮旅，我拂衣尘，君尚容辉故。永忆临歧曾怨慕：“ 东风且伴闲花住。”

昵昵当时恩怨语，话到苍茫，鬓影年华误。露电十年方旦暮，宿愁销尽新愁补。

 

良夜晤歌多感慨：“ 胡不相思，胡不相亲爱。不信春风愁不解，不相见亦长相待。”

歌烈浮生惊岁迈，书札平安，梦寐劳光彩。花雪十年愁作海，与君俱在春风外。

 

[ 蝶恋花]

逐梦逐身都忆忆，络雨联风，同是天涯客。花信在望春在侧，故情矫首了无隔。

华萼华年应惜惜，兼忘江湖，况乃东波掷。衮衮心犹沉日赤，茫茫鬓竟孤云白。

 

[ 天香.与友约咏菊花脑]

擢秀盈筐，撷芳盈袖，平生早负疏俊。清露行藏，斜阳身世，相值风期霜讯。采采春华，终未识、离离秋恨。材不材间荣悴，味无味时畦畛。

对花十年一瞬。怅中宵、冷香成阵。百感故情休论，故人休问。摇落江湖岁晚，况去日堂堂梦无准。杯酒如逢，白衣青鬓。

 

[ 踏莎行.闲君赠手制小熊泥偶并苏、王明信片]

熊梦苍凉，花身僝僽，世情幸较缨尘厚。适从屑屑养天真，陶陶佳趣托泥偶。

苏子神腴，獾郎骨秀，闲君亦自崚嶒手。平生何以报相知，论交惟欠一杯酒。

 

[ 踏莎美人.得林君赠玫瑰]

纫素于襟，簪秋入鬓，抱香勘破中肠闷。不须沧海忆斜晖，且唱《红玫瑰》复《白玫瑰》。

风露清愁，花蛾孤恨，沉沉梦与奇情问。一无言语立尘埃，直待一身如月独归来。

 

[ 踏莎美人.榴花]

一种矜红，三生清怨，温黁梦迥双双愿：人间第一是愁香，第二情怀少日动辉光。

到眼风英，劫身阳焰，幻同花我离披渐。且容烟雨擅盈盈，终胜长宵落落月孤明。

 

[ 烛影摇红.廿九生辰前夜梦渡江而返，梦中亦惊岁迈如流]

鬓入江云，襟连江树青犹隽。问愁江水复江风，影事从头幻。辜负尘涯杯卷。怅残宵、兴孤梦险：晚花冷淡，细草无情，斜阳深婉。

我亦中年，春痕不上秋眉眼。微笺何必祝长生，天意高难眷。万一人间相见。况江声、如歌如愿：一身哀乐，十载深衷，三生未远。

 

[ 喝火令.  眸君赠胭脂]

我隔沧浪水，君耽京洛缁。十年绿鬓绝新知，抛却红笺红豆，何物擅相思。

愁艳三生病，春襟万里期。天涯清露一身归，清梦离离，清泪一丝丝。清忆有情世界，犹自讳言痴。

 

[ 风入松.夜梦与岛君泛舟水上，万事浮沤，远歌微茫] 

远闻江有不维舟，世有少年游。酣歌泛剡孤光晚，觉昔时意气全收。旧雨不曾销悔，新荷已过人头。

逝波相困酒相酬，吾道在清流。溯洄烟景风湍剧，便此生许尔同愁。只是一身似月，如何两鬓都秋。

 

[ 侍香金童.得任之姐赠香炉并金刚结] 

宝焰红微，近鬓惊孤白。寸灰尽、莲经新写毕。骨冷魂清愁第一。第一炉香，爇销业力。

忏几生、恨草啼花都寂寂。怕此世、心尘犹幂幂。湖海退身劳梦役。无分禅天，有情相得。

 

[ 鹊踏花翻.得赠“玉兰小鸟”团扇感而赋]

苏绣齐纨，斯晨永夜，人间万古知荣谢。无因醉把相思，柔握馨香，泠泠欲共浮飙话。鱼虫花鸟且裴回，水情山意犹潇洒。

罢罢，鬓雪逢秋不化。异时风露翩然下。何事障面流云，合欢飞梦，入怀明月昭昭也。君心尚尔未淹留，自捐恩义后来者。

 

[ 水调.近日多梦，难名其状】

我与我鏖战，相斫卧寒僵。人间倚伏，踟蹰未死意颉颃。海岱辞锋流辈，浮世劳歌吾党，清露识行藏。梦败古人语：“慎勿作芬芳。”

兹夕永，惊故故，去堂堂。风潮四面，万虑皎皎激中肠。愿照尘骸犀火，更判春心泥絮，峭冷固其常。千载隔晨矣，斯雨转新阳。

 

[ 唐多令.牵牛花]

寒稚露华沉，妍催日色曛。想佳辰、都在斯晨。梦远披衣移烛夜，空省记、紫罗裙。

花亦有情身，香如到眼尘。酝秋云、酿作秋魂。唯是盈盈终不语，还得似、我和君。

 

[ 采桑子.立秋]

奔雷思蕴危盟誓，稠夜幽幽，浮梦休休，江海论心碧未酬。

余怀予美惊何在，追少年游，赢小淹留，雨到风檐鬓到秋。

 

采兰搴蕙空谁与，昔鬓成云，今鬓同尘，冲雨停杯各自珍。

无多新涕华年警，昔我花身，今我何人，磊落中宵一问君。

 

[ 南乡子.感事]

四野路歧长，危涕循循乌有乡。大道悲麟安足恃，红桑，海簸云迁不可详。

斫地愧时康，恤我何曾材豫章。零露漙兮生弱草，低昂，燹火秋风一并凉。

 

[ 翻香令.得小倾赠香]

晚华幽泛小崇兰，忆君明月在眉端。韶华半，春尘满，有襟灰冉冉已生烟。

一杯风露意波澜，宛然古涩破苍寒。梦清晤，何年共，要衔花披雨到空山。

 

[ 荷华媚.六百年莲子开花]

空江涉初晚。浮花气、古梦新愁都断。相思烟月改，云埋水隔，共东风泪满。

六百载、南浦西洲别，料清秋碧冷，斜晖红怨。还容我、深深愿：沧波应照见，旧时眉眼。

 

[ 鹧鸪天.屏前感赋]

秋水停云欲夜时，小山歌酒杳何之。诸星寂历陈三愿，群卉纷华拣一枝。

青眼客，白头诗，人生相得替相思。风波迢递连年少，日月于征各自知。

 

与诸友戏言建“飞熊社”，约以“我是人间一只熊” 为结

[ 鹧鸪天.归家江上有感]

牵迹衣尘化北风，岁云雌伏大江雄。擘波丸日红无谓，蹈夜旌心碧有穷。

身汩没，鬓霜蒙。故园秋隔泣鱼龙。十围故树中庭老，我是人间一只熊。

 

[ 西溪子]

隔阔音尘深怖，积梦江湖同赋。役飞光，销影事，捐身世。冰炭怀中潜置。仰雨峙新秋，自摩愁。

 

[ 寿阳曲]

花逾足，风豁眸，雨当眉，劫波身后。心兵载愁靡不覆，及言深，梦驯情囿。

 

[ 桂殿秋.桂花新开]

无限月，有垠秋。歌成鬓改与之游。一香劫动吹人世，天地腾腾犯夜稠。

 

[ 桂华明.同上]

裛香心上辞还住，衣上朱非素。两眉清气一肩秋，殢昨夕、微哉语。

花光照眼愁幽阻，照夜星芒古。不迷烟水况吴霜，有明旦、危哉雨。

  

  

诗：

 

[ 枇杷]

庭中孤秀植奇树，出阶低檐花成瀑。繁枝不用乌衔来，狸奴高踞春昼午。徙倚前看碍叶深，打头长讶湛湛露。五月离离垂朱实，手摘露彩犹滴沥。火枣交梨各有适，清芬皓齿荐嘉客。岂独薄醪消块垒？何意人间乖良会？馨香急趁旧东风，不辞缄封三百里。感得芳贻飞光快，故情新兴犹相待。日魂初破星髓改，自有团团明月在。枇杷如月宵如海，坐啖劳生憎与爱。

 

[ 屏前口占]

曾向沧波题姓字，曾徂丘壑酬微志。劳歌浮世意多违，不负天涯三二子。

 

[ 廿九口号]

桑海栖迟三宿身，意中星象远良辰。百年迹往痴文字，一带潮生空酒樽。幸未得名同竖子，时惟弹泪许畸人。微心浊世妨犹契，到此尘骸思不存。

 

[ 感赠]

故人赠我新碗碟，烟岚色成剜明月。泠泠春水破春冰，瑟瑟乱云驻乱雪。翠斝金樽饰妍词，坚白殊禀珪璧姿。红埃百战梦料峭，犹值静夜传素瓷。缄情惘惘托唐草，隔花千年风露晓。至今幽绪犹蔓生，浮氲坐觉双鬓老。且喜佳器皎如日，荐茶载醪荡忧栗。且恨长安远如日，岁晚加餐须相忆。感君用心至精洁，不独花萼有清辉。愧我孤怀气苍然，不独天末有余悲。

 

[ 病中口占]

意气良图各弃捐，蹉跎时命讳徂年。居安丈室维摩病，投老樊南风雨篇。一梦新曾赎恩义，百骸何计脱缠绵。我今伏枕艰难遍，强乞人间事事圆。

 

[ 玉兰]

冉冉庭中木莲花，怀芳心苦思无邪。敷荣春际疑海气，含辉浩叹殊天涯。寥天日华愁照灼，行云飞语惊斑驳。娟娟隔望欲生烟，昔时情素阻佳约。馀妍向晚暗雨深，馨香离披减清阴。一枝擎出轻尘外，江湖颜色梦销沉。嗟哉幽奇误玉兰，不共狂花曲阑干，一生形迹没风传。嗟哉吾辈俱中年，尘寰花海倍觉艰，愧惜微躯载覆间。但仰孤明层顾指：幸未得名同竖子。劳生汲汲皆闲事，朝闻花开夕可死。

 

[ 熬夜赶材料口占]

文章微婉雨连空，独夜风雷纸上雄。海内尽闻壮心死，也无人赋许熊熊。

 

[ 戏言画玉兰不可无诗]

辛夷花，迟放蕊，用心紫，目想幽奇盈一纸。得似万草承天露，孤华峭拔任裂素。得似三山势崚嶒，墨崩彩骞俱不朋。冰笺霜缣各良工，色铅顿挫亦摧锋。转腕历乱适我愿，自泯真象恣狂踪。欹倾骤雨临笔端，豫愁风叶颤春残。翻疑故人鬓飒飒，众中应作欢喜看。

 

[ 感事]

维新维旧瞽重瞳，砺带河山梦不同。上德由来矜若谷，壅川已自起悲风。

 

王业艰辛方演梦，上林前日正承恩。百年赋命多奇悔，弃我何曾追道存。

 

[与人戏咏猪蹄]

醇酒妇人只强欢，韭黄薤白洽杯盘。闻韶未识鱼庖愧，教子犹欺狸梦残。燕市愁穷留剧饮，鸿门剑吷劝加餐。豚蹏不抵头颅好，髀肉生涯各自安。

 

[无题]

结梦香云孤隐沦，纷拿踞世海扬尘。时余逃地衣冠客，了绝藏锋抱器人。哀近河山安质朽，歌同日月戴恩新。春途百战缨华旧，辔夜桃花独可亲。

 

[无题]

衔杯衔咒偃心兵，浪说嵚崎夜锦行。万镜何曾泯言象，两人元不及生平。匏瓜悲渐灵犀匹，鱼沫竭从梦泽盈。大患江湖犹寄命，此身堪老未堪情。
